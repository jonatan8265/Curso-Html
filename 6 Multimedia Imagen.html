15. Imagenes Con Etiqueta Figure y figurecaption

El elemento HTML figure representa contenido independiente, a menudo con un título. Si bien se relaciona con el flujo principal, su posición es independiente de éste. Por lo general, se trata de una imagen, una ilustración, un diagrama, un fragmento de código, o un esquema al que se hace referencia en el texto principal, pero que se puede mover a otra página o a un apéndice sin que afecte al flujo principal. Este es un contenido complementario, donde tambien se puede insertar imagenes.

la anterior frase esta dentro de una etiqueta figcaption es etiqueta hijo de etiqueta figure
